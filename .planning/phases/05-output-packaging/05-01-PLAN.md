Add a `write_enriched_excel(ideas, enrichments, output_path)` function to enrich_calendar.py (place it after the enrich_row function, before __main__).

Parameters:
- ideas: List[ContentIdea] -- the original loaded ideas
- enrichments: List[Dict[str, Any]] -- list of enrichment dicts returned by enrich_row()
- output_path: str -- path to write the output Excel file

Column layout (grouped by concern, in this exact order):
1. INPUT GROUP: Day, Date, Type, Topic, Description
2. QUERY GROUP: Topic Keywords, Search Queries
3. EXAMPLE 1: Ex1 URL, Ex1 Views, Ex1 Likes, Ex1 Comments, Ex1 Shares, Ex1 Author, Ex1 Caption, Ex1 Audio, Ex1 Hook Summary
4. EXAMPLE 2: Ex2 URL, Ex2 Views, Ex2 Likes, Ex2 Comments, Ex2 Shares, Ex2 Author, Ex2 Caption, Ex2 Audio, Ex2 Hook Summary
5. EXAMPLE 3: Ex3 URL, Ex3 Views, Ex3 Likes, Ex3 Comments, Ex3 Shares, Ex3 Author, Ex3 Caption, Ex3 Audio, Ex3 Hook Summary
6. AUDIO GROUP: Audio Title, Audio Author, Audio Confidence, Audio URL, Audio Fit Reason
7. LLM GROUP: Remix Ideas
8. STATUS GROUP: Enrich Status, Enrich Reason

Implementation details:
- Use `from openpyxl import Workbook` (already imported as load_workbook, add Workbook import at top)
- Also import `from openpyxl.styles import Font, Alignment` for header styling
- Date column: normalize all dates to YYYY-MM-DD string format using `idea.date.strftime("%Y-%m-%d")`
- Day column: derive from idea.date using `idea.date.strftime("%A")` (Monday, Tuesday, etc.)
- Topic Keywords column: join the keywords extracted from topic+description with ", " separator. Reuse the extract_keywords logic from generate_queries (extract meaningful words >3 chars, excluding stop words). Call it inline or add a small helper.
- Search Queries column: join enrichment["queries"] list with " | " separator
- For example fields (ex1, ex2, ex3): pull from enrichments[i]["examples"] list. If fewer than 3 examples exist, leave those columns empty string.
- Remix Ideas: the enrichment dict already has this as a multi-line bulleted string. Set cell alignment wrap_text=True for this column.
- Match ideas to enrichments by row_number field.

Header styling:
- Bold font on row 1: `Font(bold=True)`
- Freeze panes at A2: `ws.freeze_panes = "A2"`
- Auto-fit column widths: iterate columns, find max content length (capped at 50 chars width), set column width to max(len(header), max_content_len) + 2. Use `ws.column_dimensions[col_letter].width`.
- Use openpyxl.utils.get_column_letter for column letter conversion.

The function should:
1. Create a new Workbook
2. Write header row
3. For each idea, find matching enrichment by row_number, write all columns
4. Apply header styling
5. Auto-fit widths
6. Save workbook to output_path

Do NOT use pandas. Use openpyxl Workbook directly (consistent with Phase 1 decision).

<task type="auto">
  <name>Task 2: Implement build_run_log function</name>
  <files>enrich_calendar.py</files>
  <action>
Add a `build_run_log(ideas, enrichments, input_path, output_path, start_time, end_time)` function to enrich_calendar.py (place it after write_enriched_excel, before __main__).

Parameters:
- ideas: List[ContentIdea] -- original loaded ideas
- enrichments: List[Dict[str, Any]] -- enrichment results from enrich_row()
- input_path: str -- path to input Excel file
- output_path: str -- path to output Excel file
- start_time: float -- time.time() at run start
- end_time: float -- time.time() at run end

Return value: Dict[str, Any] -- the complete run log structure

Run log structure:
```json
{
  "run_summary": {
    "timestamp": "2026-02-08T14:30:00Z",
    "input_file": "Content ideas.xlsx",
    "output_file": "Enriched Content ideas_2026-02-08.xlsx",
    "total_rows": 30,
    "status_counts": {
      "ok": 25,
      "partial": 3,
      "skipped": 1,
      "error": 1
    },
    "duration_seconds": 145.2
  },
  "rows": [
    {
      "row_number": 2,
      "content_type": "Story",
      "topic": "Building MVP",
      "queries": ["founder story", "building mvp", ...],
      "total_results": 87,
      "scored_results": 62,
      "chosen_audio": {
        "title": "Inspirational Beat",
        "confidence": "high"
      },
      "example_urls": [
        "https://tiktok.com/@user1/video/7001",
        "https://tiktok.com/@user2/video/7002",
        "https://tiktok.com/@user3/video/7003"
      ],
      "enrich_status": "ok",
      "enrich_reason": ""
    }
  ]
}
```

Implementation:
- Use `datetime.datetime.now(datetime.timezone.utc).isoformat()` for timestamp
- Count status values by iterating enrichments and tallying enrich_status
- Duration: `end_time - start_time`, round to 1 decimal
- Per-row: extract from enrichment dict -- queries, total_results, scored_results, audio title+confidence, example URLs (up to 3), status, reason
- Handle missing enrichments gracefully (if a row errored before enrich_row returned, it might have a partial dict)

Also add a small helper `save_run_log(run_log, log_path)` that writes the dict to JSON with `json.dump(run_log, f, indent=2, default=str)`. The `default=str` handles any datetime objects that slipped through.
