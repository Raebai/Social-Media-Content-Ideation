Add two functions after process_results() and before `if __name__ == "__main__":`:

**1. select_top_examples(scored_results: List[Dict[str, Any]], count: int = 3) -> List[Dict[str, Any]]**

Takes the already-sorted (descending by final_score) output from process_results().

For each of the top `count` results (or fewer if less available), extract:
```python
{
    "url": video["url"],
    "views": video["views"],
    "likes": video["likes"],
    "comments": video["comments"],
    "shares": video["shares"],
    "author_username": video["author_username"],
    "create_time": video["create_time"],
    "caption": video["caption"],
    "audio_title": video.get("audio", {}).get("title", ""),
    "final_score": video["final_score"]
}
```

Return list of example dicts. If scored_results is empty, return empty list.

**2. select_audio(scored_results: List[Dict[str, Any]], examples: List[Dict[str, Any]]) -> Dict[str, Any]**

Selects recommended audio track from top results.

Step 1 - Count audio occurrences among top 20:
- Take scored_results[:20] (already sorted by final_score desc)
- For each video, get audio_id from video["audio"]["audio_id"]
- Skip empty/falsy audio_ids
- Use collections.Counter to count occurrences

Step 2 - Select audio:
- If Counter has entries, pick most_common(1)[0] -> (audio_id, count)
- Find first video in scored_results[:20] that has this audio_id to get full audio metadata
- If Counter is empty (no valid audio_ids in top 20), fall back to examples[0] audio if examples exist

Step 3 - Determine confidence (AUD-03):
- count >= 3: "high"
- count == 2: "medium"
- count == 1 or fallback used: "low"

Step 4 - Build return dict (AUD-04):
```python
{
    "audio_title": audio_title,
    "audio_author": audio_author,
    "audio_id": audio_id,
    "audio_url": audio_url,
    "audio_confidence": confidence
}
```

If no audio found at all (no results, no examples), return dict with empty strings and confidence "low".
