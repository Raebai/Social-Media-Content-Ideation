Key Phase 3 decisions (all documented in STATE.md):

- Keep videos with missing create_time — avoids false negatives
- Relevance boost capped at +0.2 — prevents keyword stuffing bias
- Recency boost tiers (14d/30d) — balances TikTok algorithm preferences
- Audio selection from top 20 using Counter — better pattern detection
- Three-tier confidence: high (>=3), medium (2), low (1/fallback)
- Fallback to example 1 audio when no repeats found
